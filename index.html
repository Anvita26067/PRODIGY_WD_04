Project Details

Food Delivery App - Benzo

The Benzo project aims to create a web application that allows users to order food online from local restaurants. The application includes a menu of available dishes, options to add items to the cart, and functionality for ordering food to be delivered to the user's home.

Food Donation App - Healthy Bite

Healthy Bite is designed to reduce food wastage by supplying surplus food to needy individuals who are unable to acquire it. The app creates a system to save unused cooked food from being wasted by connecting kitchens with those in need.

Pest Detection Using Deep Learning

This project focuses on developing a robust pest detection system using convolutional neural networks (CNNs) on annotated image datasets. The goal is to create an adaptable solution capable of accurately identifying pests across various environments, aiding farmers and land managers in timely pest control interventions to promote sustainable agriculture practices.

Weather App

The Weather App is a web application built using HTML, CSS, and JavaScript. It provides real-time weather updates for various locations, allowing users to search for a city and get current weather information including temperature, humidity, and weather conditions.

Stopwatch

The Stopwatch project is a simple web application created using HTML, CSS, and JavaScript. It features start, stop, and reset functionalities.
